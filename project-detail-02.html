전기자동차 정보 사이트

+ Project Descriptions #02

React Redux를 사용하여 전기차 정보를 보여주는 웹 애플리케이션

카테고리에 알맞은 카드 이미지 노출하여 보여주고 포스트 검색 기능,카드 클릭 시 모달 오픈 하여 보여준다.

  • Website : Tamea-ev
  • Tech Stack
  • React, React Hooks, React Router, Axios
      Styled-components
  • TEAM MEMBER : 1인 개발 - 이초롱
  • Development period : 2023 , 10

- Menu bar(햄버거 아이콘 클릭 이벤트) -

네비게이션 바의 열림/닫힘 상태를 나타내는 상태 변수 isNav와 값을 변경하는 함수 setIsNav를 선언합니다.

햄버거 아이콘을 클릭할 때 호출되는 함수인 onToggle을 정의합니다. 이 함수는 현재 상태를 반전시켜 메뉴를 열거나 닫습니다.

dataMenu 배열을 순회하며 각 메뉴 항목 렌더링하고, React Router의 Link 컴포넌트를 사용하여 링크를 추가합니다.

사용자가 햄버거 아이콘 클릭시 상태에 따라 클래스를 추가하여 메뉴가 보이거나 감춰지도록 합니다.

- 카테고리에 알맞은 카드 이미지 노출 -

onCategory 함수는 사용자가 카테고리를 선택할 때 해당 카테고리에 속하는 카드만 화면에 표시하고, 필요한 상태 변수를 업데이트하여 렌더링을 관리합니다.

- 포스트 검색 기능 -

사용자가 입력한 검색어를 기준으로 데이터를 필터링하여 검색 결과를 showData에 저장하고, 검색 결과의 여부를 empty로 표시합니다. 검색 결과가 없을 때는 모든 카테고리의 항목을 보여줍니다.

- 카드 이미지 6개씩 노출 -

onView 함수는 "더보기" 버튼을 클릭할 때마다 화면에 표시되는 카드 개수를 6개씩 늘려주며, 모든 데이터를 노출했을 때에는 해당 카테고리의 모든 데이터를 보여줍니다.

-카드 클릭 시 모달 오픈 & 좋아요 개수 카운트-

특정 카드 아이템을 클릭하면 해당 카드 아이템을 모달 창에 표시하고, 모달 창을 열거나 닫는 데 사용

"좋아요" 버튼을 클릭하면 해당 카드 아이템의 "좋아요" 상태와 수를 변경하고, 아이콘의 표시 여부를 업데이트합니다.
