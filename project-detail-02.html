+ Project Descriptions #02

친구들과 소통하며 라이프 스타일을 공유하는 추억의 싸이월드 재현

누구나 자신만의 미니 홈페이지를 만들어 꾸밀 수 있고, 자신의 이야기를 담을 수 있는 SNS 서비스 앱

데이터 관리 : Local Storage를 활용하여 데이터를 클라이언트 측에서 관리하고, 서버와의 통신 없이 필요한 정보를 로컬스토리지에서 저장하고 가져옵니다.

  • Website : Team Project_cyworld
  • Tech Stack
  • HTML, CSS, JavaScript, React Router
        Redux, Styled components,Figma
  • TEAM MEMBER : 조현아, 이초롱, 김영은, 우동훈
  • 담당 기능 : 뮤직 상점 홈페이지 디자인 및 페이지 구성, 프로필 페이지, 방명록 페이지
  • Development period : 2023 , 8

- 공통 영역 -

전체 페이지 구조 설계 및 구현 - 라우터 활용

메인 페이지, 도토리 상점 페이지, 미니 페이지 레이아웃 CSS 디자인

미니홈피 gnb 메뉴 기능 구현

- MusicBox 메인 -

인기 음악 차트를 보여주는 컴포넌트입니다. useEffect를 이용한 데이터를 가져와 getMusicDate 액션을 dispatch하여 음악 데이터와 날짜를 가져옵니다.

검색어를 입력하여 음악을 검색할 수 있습니다.

장바구니를 보이거나 숨기는 기능을 가지고 있습니다.

검색된 음악 리스트를 MusicBoxList 컴포넌트로 전달하여 표시합니다.

선택된 음악의 상세 모달을 보여주는 기능도 있습니다.

- MusicBox 검색 -

검색어에 따라 필터링된 음악 리스트를 표시하는 컴포넌트입니다.

Redux 상태에서 사용자의 좋아요 정보를 가져옵니다.

검색어를 이용하여 음악 리스트를 필터링하여 표시합니다.

- MusicBox 리스트 상세 -

음악 박스 리스트의 각 항목을 나타내는 컴포넌트입니다.

음악의 좋아요 상태를 토글하고, 장바구니에 음악을 추가하거나 제거할 수 있습니다.

음악의 상세 정보를 모달로 보여주는 기능도 가지고 있습니다.

- MusicBox 장바구니 -

장바구니에 담긴 음악 리스트를 보여주는 컴포넌트입니다.

장바구니에 음악을 담거나 제거할 수 있습니다. 특정 음악 아이템을 삭제하기 위해 removeCart 액션을 dispatch합니다.

음악을 장바구니에 추가하면, 장바구니 상태를 업데이트하고 musicHistory와 addedToBasket는 local state로 관리됩니다.

- MusicBox Modal 기능 -

음악의 상세 정보를 모달 형태로 보여주는 컴포넌트입니다.

음악의 제목, 가수, 음악 비디오를 표시합니다.

방명록 페이지 - 작성 및 수정 삭제

작성한 방명록 내용을 저장하여 보여줍니다.

페이지 로드 시 기본 방명록 표시 : 초기 방명록 하나를 표시하며, 사용자 이름, 날짜, 이미지와 함께 내용을 표시합니다.

방명록 표시 및 수정/삭제 기능 : 방명록 리스트를 역순으로 매핑하여 최신순으로 표시합니다.
수정 중인 방명록의 경우 수정 내용을 입력할 수 있는 textarea를 표시합니다.
수정 및 삭제 버튼을 클릭하면 해당 방명록을 수정 또는 삭제합니다.

작성 및 수정 권한 관리 : 사용자가 로그인한 사용자인 경우에만 방명록 작성 및 수정 가능하도록 설정합니다.

프로필 페이지 - 편집 기능

'수정' 버튼 클릭 시, 편집 모드가 활성화되며 버튼은 '저장' 버튼으로 전환됩니다. 각각의 프로필 정보(이름, 성별, 생년월일, 자기소개)를 입력할 수 있는 input 요소를 표시합니다.

input 요소의 값이 변경되면 해당 값을 Redux를 통해 업데이트합니다.

현재는 사진 업로드 기능이 구현되지 않았으므로, 수정 완료 시 임시의 사진이 프로필 사진으로 설정됩니다.

- 미니홈피 배경음악 설정 기능 -

페이지 로드 시, 음악 날짜 및 재생 기록을 Redux 및 로컬 스토리지에서 가져와 표시합니다.

주크박스 배경 음악 또한 해당 기록을 관리하며, 선택 및 삭제 기능이 가능합니다. 선택된 배경 음악이 있는 경우 해당 정보를 표시하고, 없다면 "음악을 설정해주세요" 메시지를 보여줍니다.

만약 배경 음악 기록이나 해당 사용자의 기록이 없으면 "설정된 배경 음악이 없습니다." 메시지가 표시됩니다.
