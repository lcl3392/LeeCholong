Movie, 반응형 웹사이트

+ Project Descriptions #03

HTML, CSS, JavaScript, jQuery, 플러그인을 활용하여 영화 이미지 슬라이드 구현, API 가져와 영화 영상을 보여준다.

max-wirdth: 1290px, 1024px, 768px, 960px, 600px, 320px
반응형 웹사이트

  • Website : Movie_Website
  • Tech Stack
  • HTML, CSS, JavaScript, jQuery
  • TEAM MEMBER : 1인 개발 - 이초롱
  • Development period : 2024 , 01

- 배너 이미지 슬라이드 -

Swiper 라이브러리를 사용하여 영화 이미지와
제목, 간단한 설명을 보여줍니다. 슬라이드를 자동으로 재생되며 각 슬라이드가 5초 동안 보여집니다.

- 영화차트 ,탭 메뉴 -

Swiper 라이브러리를 사용하여 영화 차트 4개의 영화 이미지 노출, 슬라이드를 자동으로 재생되며 각 슬라이드가 6초 동안 보여집니다.

영화 목록의 각 항목을 클릭했을 때 해당하는 항목 영화 정보를 보여주고,초기에는 모든 영화 정보를 숨기고, 첫 번째 항목에 해당하는 영화 정보를 보이도록 설정합니다.

각각의 영화 목록 항목을 클릭하여 해당하는 영화 정보를 동적으로 전환할 수 있습니다.

- 영화 모달 영상 띄우기 -

YouTube iframe API를 활용하여 영상을 제어하는 기능을 구현

사용자가 플레이 버튼을 클릭하면 모달이 열리고, YouTube 트레일러가 플레이어를 통해 재생됩니다. 중지 버튼을 클릭하면 동영상이 정지되며, 모달이 닫힙니다.

이를 통해 사용자는 편리하게 YouTube 트레일러를 모달 창에서 간편하게 관람할 수 있습니다.

- 공지사항, 탭 메뉴 -

초기에는 모든 공지사항 내용을 숨김으로 설정하고, 활성화된 탭의 내용만 보여줍니다.

사용자가 탭을 클릭하면 해당 탭의 공지사항이 보이게 되고, 다른 탭의 공지사항은 숨겨지는 간단한 탭 메뉴 동작 구현.

이렇게 사용자는 원하는 공지사항을 쉽게 선택하여 확인할 수 있습니다.

- 반응형 작업 -

반응형 웹 디자인은 다양한 디바이스에서 웹사이트를 일관된 사용자 경험으로 제공하기 위해 주어진 각 크기에 대한 max-width 미디어 쿼리를 사용합니다.

화면의 너비가 특정 지점을 넘어서면 적절한 스타일 및 레이아웃을 적용하여 사용자에게 최적화된 1290px, 1024px, 768px, 960px, 600px, 320px 크기를 구현하였습니다.
